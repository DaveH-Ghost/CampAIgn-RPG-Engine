# Version 1.2.1 Changelog (CampAIgn-RPG-Engine)

**Purpose:** Entity templates — save and spawn portable object/agent blueprints without ids or placement.

**Baseline:** **1.2.0**

---

## Additions

- `export_object_template(obj)` / `export_agent_template(agent, include_memory=False)`
- `spawn_object_from_template(session, template, position, area_id=...)`
- `spawn_agent_from_template(session, template, position, area_id=...)`
- `validate_template(data)` — template document validation
- `create_agent_in_area(..., allow_duplicate_name=False)` — template spawn allows duplicate display names

Template JSON uses `template_version: 1`, `kind: "object" | "agent"`, and omits runtime ids/position. Optional `memory` block on agents when exported with `include_memory=True`.

**No `snapshot_version` bump.**
